# キロ程 画像比較・描画ツール 要件定義・仕様書

## 1. 概要

鉄道軌道画像（マーキングあり／なし）を比較表示し、画像上に変状箇所（ゆるみ・空洞）や除外区間を描画・記録するGUIツール。
キロ程情報に基づいて画像をグループ化し、3つの作業エリア（左軌間外・軌間内・右軌間外）で描画を行う。
**上下線（上り／下り）と単線を区別**して管理し、同一キロ程でも線種が異なれば別画像として扱う。
Excelテンプレート方式による報告書出力、PDF出力、集計ログ出力、ヒートマップ表示機能を備える。

## 4. 画面構成

### 4.1 メインウィンドウ (1920x1000)

- **コントロールパネル（上部）**: フォルダ選択、Excel出力、PDF出力、集計表出力、除外CSV一括取込、ページ送り（±1/±10）
- **ツールパネル**: 入力モード/移動モード切替、形状選択（四角/丸）、種別選択（ゆるみ/空洞/除外区間）
- **画像表示エリア（中央）**: マーキングあり/なしの左右同期キャンバス（Y=900pxでトリミング表示）
- **キロ程表示**: `キロ程：下り 172k000m（13/113）` 形式。下り=青文字、上り=赤文字、単線=デフォルト色
- **下段テーブル**: 変状一覧（管理番号/種別/エリア/範囲/深さ/除外理由）、変状一覧ボタン、ヒートマップボタン、削除ボタン
- **右サイドバー**: キロ程一覧（`下_172k000m` / `上_172k000m` / `012k120m` 形式、線種で色分け）

### 4.2 変状一覧編集ダイアログ

- 現在ページの変状/除外を変状タブ・除外タブに分離表示
- 管理番号をダブルクリックで編集（入れ替え対応）
- 除外理由の編集、削除機能、保存/閉じるボタン

### 4.3 ヒートマップウィンドウ (1500x950)

- 全キロ程を連結表示（全体/左軌間外/軌間内/右軌間外の4行）
- ゆるみ（青）・空洞（赤）・除外区間（上部帯）の表示ON/OFFスイッチ
- **波形画像表示**: 各エリア行にマーキングあり/なし画像のストリップを背景表示（ON/OFFスイッチ付き）
- **画像ソース切替**: 「マーキングあり」/「マーキングなし」ボタンで表示画像を切替
- **等倍表示**: 等倍ボタンON時、波形画像のネイティブ縦横比（844:179）を維持して行高さを動的調整
- **上下線切替**: 「下り」/「上り」/「単線」ボタンで表示する線種を切替（上下線混在時のみ表示、単線ボタンは非アクティブ）
- ホイールズーム、ドラッグスクロール、マウスホバーでキロ程リアルタイム表示
- キロ程途切れをオレンジ破線で表示
- **連続波形画像Excel出力ボタン**: レイヤー選択メニュー経由でExcel出力（詳細は8.6節参照）

## 6. 描画・編集機能

### 6.1 入力モード
- ドラッグで四角/丸を描画、種別ごとの色で表示
- 描画完了時に管理番号入力ダイアログ（自動採番+編集可能、重複不可）
- 除外区間: クリックでモーダルダイアログ（開始/終了位置、管理番号、除外理由、他エリア同時入力スイッチ）

### 6.2 移動モード
- シェイプクリックでバウンディングボックス表示（4pxハンドル）
- ドラッグで移動（エリア制限付き）、ハンドルでリサイズ
- シェイプ外クリックまで選択維持（連続変形対応）
- テーブル行選択↔キャンバスシェイプ選択の双方向同期

## 7. データベース

テーブル: `drawings`

| カラム | 型 | 説明 |
|--------|----|------|
| id | INTEGER PK | 一意識別子 |
| kilo | TEXT | 複合キー（例: `d_172k000m`, `u_172k000m`, `s_012k120m`） |
| area | TEXT | エリア名 |
| shape_type | TEXT | rectangle/oval |
| lx0, ly0, lx1, ly1 | REAL | 図形座標 |
| tx, ty | REAL | テキスト位置 |
| text_lbl | TEXT | テキスト文字列 |
| location_str | TEXT | キロ程文字列 |
| category | TEXT | ゆるみ/空洞/除外区間 |
| mgmt_number | INTEGER | 管理番号 |
| exclusion_reason | TEXT | 除外理由 |

DB接続時に `location_str` を現在のフォーマットで自動再生成（マイグレーション）。
旧形式キー（接頭辞なし `172k000m`）は自動的に `s_172k000m` にマイグレーションされる。

## 8. Excel出力（テンプレート方式）

### 8.1 テンプレート
- デフォルトパス: `Q:\004_検測G\解析共用\UTRAS解析マクロ・マニュアル\アトラス報告書.xlsx`
- `temp` シートを `copy_worksheet` で複製、データ書き込み後に `temp` を削除
- テンプレートのPNG画像はBytesIOで手動コピー
- テンプレートのネイティブ図形はzip再処理で注入

### 8.2 画像配置
- PILでY=900pxトリミング、DPI=96で保存
- テンプレートと同じ `twoCellAnchor editAs="oneCell"` で配置
- アンカー座標: from(0,0,0,0) to(4,1003024,35,165652)

### 8.3 変状シェイプ
- `twoCellAnchor` でセル座標に変換して配置
- COMで取得したテンプレートの列幅・行高さ・画像表示サイズからポイント→EMU変換
- 管理番号ラベル: `txBody` 付きテキストボックス（18pt太字、種別色）

### 8.4 変状一覧
- A39〜A42（1〜4件目）、B39〜B42（5〜8件目）
- 管理番号上位8件、除外区間除外
- フォーマット: `① 右：12k008.5m ～ 12k011.0m　ー　0.6m ～ 1.3m（ゆ）`

### 8.5 出力オプション
- 範囲選択: チェックボックス方式（ドラッグ連続ON/OFF対応）
- 一括ファイル / 個別ファイル出力
- デフォルトファイル名: `解析報告書（xxkxxm～xxkxxm）.xlsx`
- フッター中央にページ番号

### 8.6 連続波形画像Excel出力（テンプレート方式）

ヒートマップウィンドウから起動する連続波形画像の一括Excel出力機能。

#### テンプレート
- デフォルトパス: `Q:\004_検測G\解析共用\UTRAS解析マクロ・マニュアル\連続波形画像テンプレート.xlsx`
- `temp` シートを複製、テンプレートの画像・ヘッダー・ネイティブ図形を保持

#### レイアウト
- **1シートあたり4行 × 100m = 400m**
- 1ブロック（100m）= 2行分の `TwoCellAnchor`（A〜D列フィット）で配置
- A3横向き

#### 合成画像の構成（1ブロック = 100m）
- **上部スケール**: 0〜20m繰り返し表示（5m毎ラベル、0mは左端揃え、20mは右端揃え）
- **左側**: エリア名ラベル + 深さスケール（0.0〜2.0m、非線形マッピング対応）
- **中央**: 各エリア（左軌間外・軌間内・右軌間外）の波形画像ストリップ
- **下部スケール**: 絶対キロ程表示（20m毎ラベル）
- **20m区切り線**: 白色、太さ2px（目盛り線より太め）
- エリア間ギャップ: 16px
- 右端マージン: 60px（目盛りラベル見切れ防止）

#### 出力メニュー（レイヤー選択ダイアログ）
- **出力線別**（上下線混在時のみ有効）: `□上り` / `□下り`（デフォルト: OFF、単線時は非アクティブ）
  - 上下線選択時はファイル分割出力（`下_連続波形画像.xlsx` / `上_連続波形画像.xlsx`）
  - 線別未選択で出力ボタン押下時は警告表示
- **波形画像**: マーキングあり / マーキングなし（デフォルト: なし）
- **出力エリア**: 左軌間外 / 軌間内 / 右軌間外（個別ON/OFF）
- **オーバーレイ**: ゆるみ / 空洞 / 除外区間（個別ON/OFF）
- **ヘッダー設定**（ON/OFF切替、デフォルト: ON）
  - 左ヘッダー: 任意テキスト（デフォルト: `番号：〇〇〇〇` / `件名：〇〇〇〇`）
  - 右ヘッダー: 任意テキスト（デフォルト: `線名/駅間　〇〇線/〇〇～〇〇間` / `計測キロ程　&A`）
  - `&A` はExcelのシート名自動挿入コード（キロ程範囲が入る）
  - 文字サイズ: 6〜72pt（デフォルト: 24pt）、フォント: Meiryo
- 保存先ダイアログの初期フォルダ: 作業中フォルダ

## 13. 上下線・単線対応

### 13.1 線種判定
ファイル名のパターンから線種を自動判定:
- `_d_` → 下り線（down）
- `_u_` → 上り線（up）
- `_s_` → 単線（single）※デフォルト

### 13.2 複合キー
画像・DB管理に使用するキー形式: `{線種}_{キロ程}`
- 例: `d_172k000m`（下り）、`u_172k000m`（上り）、`s_012k120m`（単線）
- `parse_kilo()` は接頭辞を無視して数値変換（後方互換）

### 13.3 表示・ソート
- **並び順**: 下り線 → 上り線 → 単線（線種内はキロ程数値順）
- **キロ程表示**: 下り=青文字、上り=赤文字、単線=デフォルト色
- **キロ程一覧**: `下_172k000m` / `上_172k000m` / `012k120m`（単線は接頭辞なし）
- **Excelシート名**: `下_172k000m` / `上_172k000m` / `012k120m`

### 13.4 DBマイグレーション
旧形式キー（接頭辞なし）はDB接続時に自動的に `s_` 接頭辞を付与してマイグレーション。
