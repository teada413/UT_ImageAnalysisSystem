1. はじめに

本システムは、鉄道軌道のUT検査画像（マーキングあり／なし）を比較表示し、画像上に変状箇所（ゆるみ・空洞）や除外区間を描画・記録するためのツールです。

動作要件

  • Windows 11
  • Microsoft Excel（PDF出力機能を使用する場合）

作業フォルダの構成

以下の構成でフォルダを準備してください。

作業フォルダ/
├── xxxx_マーキングあり/
│   ├── 20250927_002_両毛線_s_s_012k120m_c.jpg
│   ├── 20250927_002_両毛線_s_s_012k140m_c.jpg
│   └── ...
└── xxxx_マーキングなし/
    ├── 20250927_002_両毛線_s_s_012k120m_c.jpg
    └── ...
画像ファイル名にキロ程（例: 012k120m）が含まれている必要があります。

2. 起動と作業フォルダの選択

  1. アトラス画像解析システム.exe をダブルクリックして起動します。
  2. 画面左上の フォルダを選択 ボタンをクリックします。
  3. 作業フォルダ（マーキングあり／なしのサブフォルダを含む親フォルダ）を選択します。
  4. データベースが存在しない場合、新規作成の確認ダイアログが表示されます。「はい」を選択してください。
  5. 画像が読み込まれ、最初のキロ程が表示されます。
作業データは作業フォルダ内の drawings.db に自動保存されます。次回同じフォルダを開けば前回の作業が復元されます。

画像表示エリア（中央）

左側に「マーキングあり」、右側に「マーキングなし」の画像が表示されます。左右は完全に同期しています。

変状一覧（下段）

現在のキロ程に登録されている変状・除外区間の一覧が表示されます。除外区間の行は薄い赤色で表示されます。

キロ程一覧（右サイドバー）

読み込んだ全キロ程が一覧表示されます。クリックするとそのキロ程にジャンプします。

4. 変状の入力（入力モード）

  1. ツールパネルで 入力モード が選択されていることを確認します。
  2. 形状（四角 / 丸）と 種別（ゆるみ / 空洞）を選択します。
  3. 画像上の描画エリア（シアン色の破線枠内）でマウスをドラッグして図形を描画します。
  4. 描画完了時に 管理番号入力ダイアログ が表示されます。
    • 次の番号が自動提案されます。必要に応じて変更してください。
    • 同一キロ程内で番号の重複はできません。
    • キャンセルすると描画は取り消されます。

種別ごとの色

種別色説明
ゆるみ■ 青ゆるみの可能性がある箇所
空洞■ 赤空洞の可能性がある箇所
除外区間■ 赤（半透明）設備等による反射などの除外範囲
描画はエリア内に自動クランプされます。エリア外をクリックしても反応しません。

5. 除外区間の入力

  1. 種別で 除外区間 を選択します。
  2. 画像上の描画エリアをクリックすると、除外区間設定ダイアログが開きます。
  3. 以下の項目を入力します。
    項目説明
    管理番号自動採番（変状とは別カウント）。編集可能。
    開始位置 (m)0.0〜20.0m の範囲で指定
    終了位置 (m)開始位置以降の値を指定
    除外理由デフォルト：「設備等による反射」
    他エリアにも同時入力チェックを入れると同じ範囲を他エリアにも一括入力
  4. OKを押すと除外区間が登録されます。
同一エリア内で除外区間の範囲が重複する入力はできません。

6. シェイプの移動・リサイズ（移動モード）

  1. ツールパネルで 移動モード を選択します。
  2. シェイプをクリックすると、白い破線のバウンディングボックスと8つのハンドルが表示されます。
  3. 移動：シェイプ本体をドラッグします。エリア外には移動できません。
  4. リサイズ：ハンドル（角・辺の四角い点）をドラッグします。
  5. シェイプ以外の場所をクリックすると選択が解除されます。
バウンディングボックスはシェイプ外をクリックするまで表示され続けます。連続して変形操作が可能です。
下段の変状一覧で行をクリックすると、対応するシェイプが自動選択されます。逆に、画像上でシェイプを選択すると一覧の対応行がハイライトされます。

7. 変状一覧の操作

メイン画面の一覧テーブル

  • 管理番号：ダブルクリックで直接編集できます。
  • 除外理由：除外区間行のみ、ダブルクリックで編集できます。
  • 選択した図形を削除：選択行の図形をDBから削除します。

変状一覧編集ダイアログ

下段の 変状一覧 ボタンを押すと、現在のキロ程の変状を一括編集できるダイアログが開きます。

  • 変状タブ：ゆるみ・空洞の一覧。管理番号を編集可能。
  • 除外タブ：除外区間の一覧。管理番号・除外理由を編集可能。
  • 管理番号の入れ替え（例：①と②を交換）にも対応しています。
  • 変更後は 保存 ボタンを押してください。
  • 未保存のまま閉じようとすると確認メッセージが表示されます。

8. Excel出力

  1. Excel出力 ボタンをクリックします。
  2. 出力範囲選択ダイアログが開きます。
    • チェックボックスで出力するキロ程を選択します。
    • ドラッグで連続ON/OFFが可能です（チェックボックス上をドラッグ）。
    • 「全選択」「全解除」ボタンも利用できます。
    • 個別ファイルで出力するにチェックを入れると、キロ程ごとに1ファイルずつ出力されます。
  3. テンプレートファイルが見つからない場合、選択ダイアログが表示されます。
  4. 保存先を指定して出力します。

出力内容

  • テンプレートの temp シートを複製し、キロ程名のシートを生成
  • 上段：トリミング済み画像（Y=900px）＋変状シェイプ＋管理番号ラベル＋凡例
  • 下段：変状一覧（管理番号上位8件、除外区間は除外）
  • フッターにページ番号

デフォルトファイル名

解析報告書（012k120m～012k540m）.xlsx（選択範囲の最小〜最大キロ程）

9. PDF出力

  1. PDF出力 ボタンをクリックします。
  2. Excel出力と同じ範囲選択ダイアログが開きます。
  3. 保存先を指定すると、内部でExcelを生成しPDFに変換します。
  4. 変換中はスピナーが表示されます。
PDF出力にはMicrosoft Excelのインストールが必要です。
個別ファイル出力の場合、進捗（例：「PDF変換中... (3/10)」）が表示されます。

10. 集計表出力

  1. 集計表出力 ボタンをクリックします。
  2. 作業フォルダ内に「集計表」フォルダが作成され、各キロ程の詳細.log と集計.log が出力されます。
  3. 既にファイルが存在する場合は上書き確認が表示されます。

出力ファイル例

ファイル内容
1012540_xxx詳細.log変状・除外区間の詳細情報
1012540_xxx集計.log空洞・ゆるみの件数集計

11. 除外CSV一括取込

  1. 除外csv一括取込 ボタンをクリックします。
  2. CSVファイルが格納されたフォルダを選択します。
  3. CSVファイル名は画像ファイル名と対応している必要があります（例：20250927_002_両毛線_s_s_012k120m.csv）。
  4. 既に除外区間が存在する場合、上書き（既存削除→取込）または追加（既存維持→追加取込）を選択できます。

CSV形式

除外設定
左,5.0,6.0
左,15.5,19.5
右,19.0,19.5
内,19.0,19.5
NO.,メッシュ内割合[%],ファイル名,...
（以降は読み込み対象外）

14. トラブルシューティング

症状対処
「エラー: サブフォルダが見つかりません」作業フォルダの直下に「マーキングあり」「マーキングなし」を含む名前のサブフォルダが必要です。
Excel出力でテンプレートが見つからないデフォルトパスにテンプレートがない場合、ファイル選択ダイアログが表示されます。手動で選択してください。
PDF出力に失敗するMicrosoft Excelがインストールされているか確認してください。出力先フォルダに書き込み権限があるか確認してください。
管理番号が重複エラーになる同一キロ程内で同じ番号は使用できません。変状一覧ダイアログで番号の入れ替えが可能です。
画像が「画像なし」と表示されるマーキングなし画像が存在しない場合に表示されます。マーキングあり側は正常に動作します。
